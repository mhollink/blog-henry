# Henry Hollink

A blog page project using marked and gray-matter to talk about my stillborn son Henry.

## Blogs/posts

Blogs and posts are written in Markdown files and are located in [public/posts/](./public/posts)

| Type           | Markdown Syntax                         | Example Render                       |
|----------------|-----------------------------------------|--------------------------------------|
| **Header 1–6** | `#`, `##`, `###`, etc.                  | `# A Big Title` → `<h1>`             |
| **Paragraph**  | Just plain text                         | A normal paragraph.                  |
| **Bold**       | `**bold text**`                         | **bold text**                        |
| **Italic**     | `*italic text*`                         | *italic text*                        |
| **Image**      | `![alt text](path/to/image.jpg)`        | `<img>`                              |
| **Link**       | `[text](https://example.com)`           | [example](https://example.com)       |
| **Quote**      | `> quoted text`                         | <blockquote>quoted text</blockquote> |
| **List**       | `- item` or `1. item`                   | Unordered or ordered list            |
| **Code**       | `` `inline code` `` or triple backticks | syntax highlighting blocks           |
